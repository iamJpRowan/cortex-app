# Getting Started

## Run Cortex

Start the app from your terminal or by opening the application. The first time you open Cortex, you'll see the Home view.

## Main areas

- **Home** — Overview and quick access. Use the sidebar to switch views.
- **Chat** — Have conversations with the AI. Choose a model and start typing. Responses stream in and support markdown and code.
- **Settings** — Configure AI providers (e.g. Ollama, Anthropic), select models, and adjust appearance (theme).
